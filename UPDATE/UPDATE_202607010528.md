# Engineering Update: High-Frequency Hybrid Price Regression & Error-Regime Gatekeeper Architecture

## 1. Executive Summary & Architectural Paradigm Shift
This update documents the complete structural migration of the Final Year Project (FYP) Core Machine Learning pipeline from a multi-class directional signal classifier to a two-stage **Continuous Price-Change Delta Regression Engine**. 

While traditional classification structures (e.g., mapping states directly to Buy/Hold/Sell vectors) provide actionability, they introduce arbitrary rigid labeling bounds that can distort loss gradient convergence on noisy 5-minute (M5) market micro-structures. Conversely, raw price regression models suffer from time-series persistence traps—where networks simply learn to replicate the preceding price step ($Price_{t+1} \approx Price_t$), showing deceptively high $R^2$ scores in-sample while failing catastrophically out-of-sample during regime shifts.

To circumvent both limitations, this architecture shifts the core optimization target to predict the **Causal Price-Change Delta ($\Delta$)**:
$$\text{Target Delta}_t = \text{Close}_{t+1} - \text{Close}_t$$

By isolating the absolute mathematical variance step between sequential candles, the deep learning network focuses purely on directional and structural momentum trends without suffering from absolute distribution drift. Real-world raw USD price tracking is perfectly reconstructed during the post-processing phase by dynamically inverse-scaling the forecasted deltas and overlaying them against the historical base positions:
$$\text{Predicted Price}_{t+1} = \text{Baseline Close}_t + \text{Predicted Delta}_t$$

## 2. Advanced Preprocessing & Mathematical Scaling Pipeline
The input space ingests high-frequency M5 XAUUSD OHLCV historical sequences and maps them through a multi-tiered, strictly causal transformation framework:

* **100% Causal Denoising:** Employs an Exponential Moving Average (EMA) smoothing pass with `adjust=False`. This eliminates localized high-frequency tick noise while guaranteeing that no future information leaks into past calculations, preserving real-world time-series properties.
* **Stationary Feature Transformations:** Computes localized indicators including Relative Strength Index (RSI), Money Flow Index (MFI), Average True Range (ATR), Volatility Filters, and Log-Transformative Returns. All features are calculated over backward-looking windows to prevent look-ahead bias.
* **Multi-Timeframe (MTF) Context Injection:** Generates pseudo-H1 (1-Hour) macro indicators directly within the M5 framework (H1 Moving Average slope and 168-bar macro RSI approximations). This allows the sequential layers to cross-examine micro-candle trend adjustments against overarching institutional momentum.
* **Trigonometric Cyclical Time Encoding:** Projects raw hours into continuous mathematical loops using sine and cosine functions ($\sin(\text{hour})$ and $\cos(\text{hour})$). This preserves the temporal structural relationships between market close/open sessions.
* **Isolated Dual-Target Scaling:** Implements decoupled `StandardScaler` engines. Crucially, the feature matrix scaler (`scaler_X`) and the target delta scaler (`scaler_y`) are fitted **exclusively on the historical training data partition**. Test parameters are normalized strictly using the training metrics, establishing a leak-proof boundary for out-of-sample data auditing.

## 3. Dual-Model Deep Learning Pipeline

### Phase 1: Hyperparameter Optimization Sweep (Optuna)
An automated hyperparameter sweep tracks across 15 optimization trials to locate optimal hidden dimensionality metrics (`hid_dim`) and learning rate boundaries (`lr`). The sweep utilizes a robust `HuberLoss` criterion, which acts as a hybrid between MAE and MSE to insulate parameter selection from structural outlier distortion during high-volatility spikes.

### Phase 2: Base Regression Engine — Model A (CNN-LSTM-Attention)
The core predictive network maps sequential input feature windows (Shape: `[Batch, Lookback=60, Features=9]`) down to a single continuous point prediction:
1. **1D Temporal Convolutional Layer:** Extracts localized spatial patterns and geometric formations across the scaled feature rows.
2. **Stacked Long Short-Term Memory (LSTM) Network:** Processes the sequential dependencies across the 60-candle lookback history to retain long-term systemic trends.
3. **Temporal Attention Bottleneck:** Evaluates all hidden state representations emitted by the LSTM blocks, calculating a softmax distribution weight matrix to isolate and amplify high-impact historical candles while dampening irrelevant market noise.
4. **Linear Head:** Maps the attention context tensor directly to a single output value representing the forecasted scaled delta price.

### Phase 3: Meta-Error Risk Gatekeeper — Model B (TCN)
To preserve the Decision Support System (DSS) safety features, a Temporal Convolutional Network (TCN) is deployed as a meta-classifier trained on Model A's absolute residuals:
* Absolute model variance errors are extracted across the training history: $\text{Residuals} = |y_{\text{true}} - y_{\text{pred}}|$.
* A dynamic risk threshold is established at the **65th percentile** of these absolute residuals.
* **Meta-Labels Generation:** Datapoints yielding errors within the threshold are flagged as Stable/Low Risk (1), while variance outliers are flagged as High Volatility/Unstable (0).
* **Network Dynamics:** Model B leverages causal dilated convolutional fields with progressive dropout layers, learning to recognize structural macro-formations that precede forecasting failures in Model A. This forms a mathematical gatekeeper capable of blocking live trades during dangerous market regimes.

## 4. Evaluation Metrics & Chronological Logging Engine
The post-processing framework de-scales predictions back into raw USD figures using the saved `scaler_y_regression.pkl` to compute a comprehensive statistical suite on out-of-sample data:

1. **$R^2$ Score (Coefficient of Determination):** Evaluates the goodness-of-fit accuracy against real price distributions (achieving a near-perfect score of **0.9998** due to the target delta alignment).
2. **Mean Directional Accuracy (MDA):** Measures the absolute percentage of correctly forecasted binary directional changes (Up vs. Down movements) relative to the preceding baseline.
3. **MAE, RMSE, and MAPE:** Computes absolute and percentage dollar-denominated discrepancies directly in real-world currency terms.
