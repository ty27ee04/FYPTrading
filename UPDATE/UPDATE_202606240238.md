# Deep Learning Architecture Upgrade: V2.0

## Overview
This update introduces three major structural upgrades to the Deep Learning (DL) training pipeline (`gemini-training.py`). These changes are designed to improve the model's global accuracy, execution precision, and mathematical risk-to-reward edge in the XAUUSD market.

## What was Updated?
1. **Focal Loss Implementation:** Replaced the standard PyTorch `CrossEntropyLoss` with a custom `FocalLoss` module.
2. **Asymmetric Triple Barriers:** Shifted the risk-reward structure in the Triple Barrier Method from symmetric (1:1) to asymmetric (1:1.5).
3. **Multi-Timeframe (MTF) Context Injection:** Added Higher Timeframe (H1) technical metrics directly into the M5 input sequence.

## How it was Updated
* **Focal Loss:** A new class `FocalLoss(nn.Module)` was created. It utilizes an `alpha` array `[0.2, 1.0, 1.0]` to heavily down-weight the massive majority class (`Hold = 0`), and a `gamma=2.0` focusing parameter to exponentially penalize the model for missing difficult `Buy` or `Sell` signals.
* **Barriers:** In `preprocess_gold_data`, the default arguments were updated from `pt_mult=2.5, sl_mult=2.5` to `pt_mult=3.0, sl_mult=2.0`.
* **MTF Features:** Since 1 hour equals 12 M5 bars, rolling calculations were added to compute the `ma_h1` (12-bar moving average), `h1_trend_slope`, and `rsi_h1` (approximated over 168 M5 bars). These were appended to `feat_cols`, dynamically scaling `in_dim` from 7 to 9.

## Why it was Updated
* **Solving the Class Imbalance Trap:** Standard CrossEntropyLoss causes the neural network to "cheat" by predominantly guessing "Hold" since financial markets are mostly sideways noise. Focal Loss solves this mathematically by forcing the gradient updates to focus almost entirely on the trend reversals.
* **Positive Expectancy (EV):** Symmetric barriers require a win rate above 50% to be profitable after spreads. By skewing the barriers to 3.0 (Profit) and 2.0 (Stop Loss), the strategy enforces a 1:1.5 Risk-to-Reward ratio. The model is now structurally hunting for asymmetrical payouts, meaning it can achieve profitability even if the win rate drops to 45%.
* **Filtering Micro-Noise:** M5 price action generates frequent false breakouts. By injecting H1 trend slopes and H1 RSI into the Attention layer, the model gains "macro-awareness," allowing it to reject M5 buy signals if the H1 trend is in a strong decline.
